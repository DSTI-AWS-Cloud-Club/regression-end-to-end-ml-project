# Phase 3 - Full integration
Full integration (original Anas Riad project) -> Integrate the whole End to End pipeline.

## Steps

Clone the repository

```
uv init

uv python install 3.11

uv python pin 3.11

uv add great-expectations mlflow fastapi streamlit xgboost lightgbm scikit-learn pandas numpy matlplotlib evidently 

```

Activate environment

On Windows:
`.\.venv\Scripts\activate.ps1`  

On Linux:

`source .venv/bin/activate`
